# 🌾 Farm Labor Management System

## 📌 Project Overview
The **Farm Labor Management System** is a web-based application designed to connect farmers and laborers on a single platform. Farmers can post agricultural jobs with complete details, and laborers can easily view and accept available jobs without difficulty.

This system helps reduce communication gaps, saves time, and makes the hiring process more efficient and transparent.

---

## 🎯 Objectives
- To provide a simple platform for farmers to hire laborers
- To help laborers find job opportunities بسهولة (easily)
- To improve communication between farmers and workers
- To maintain job records and history for future reference

---

## 🚀 Key Features

### 👨‍🌾 Farmer Dashboard
- Register and Login as Farmer
- Create and Post Jobs with details:
  - Work type (Harvesting, Planting, etc.)
  - Number of workers required
  - Salary (per day / per hour)
  - Land area (in acres)
  - Crop/Work description
  - Location
- View all posted jobs
- Track which laborers accepted the job
- Delete jobs if cancelled/postponed
- View job history

---

### 👷‍♂️ Laborer Dashboard
- Register and Login as Laborer
- View all available jobs (without searching)
- Search/filter jobs if needed
- Accept jobs بسهولة
- View accepted jobs
- Track job history

---

### 👤 Profile Management
- Separate profiles for Farmer and Laborer
- Profile includes:
  - Name
  - Phone Number
  - Location
- Profile icon shows correct user details based on login type

---

### 🔐 Authentication System
- Separate login/register for Farmer and Laborer
- Password validation
- Reset password option

---

### 📊 Job Management
- Real-time job listing
- Job acceptance tracking
- Job history maintained
- Progress tracking for jobs

---

### 💾 Data Storage
- Uses **Browser LocalStorage**
- Data remains saved even after page refresh
- Data resets only on logout

---

## 🛠️ Technologies Used
- **HTML** – Structure
- **CSS** – Styling & UI Design
- **JavaScript** – Functionality & Logic
- **LocalStorage** – Data persistence
